# Solo Strength Studio

A lightweight, offline-first training companion for planning, running, and logging solo weight sessions. Everything runs in the browser and persists to `localStorage`, so your data stays on your device.

## Features

- **Program Builder** – Create custom training days with exercises, set/rep targets, and notes.
- **Active Session Mode** – Load a day, mark sets complete, tweak weights/reps, and quick-log results.
- **Rest Timer** – Built-in 2-minute countdown with start/pause/reset controls.
- **Training Journal** – Store detailed session entries with dates, loads, and highlights.
- **Progress Snapshot** – Visualize your heaviest logged lifts per exercise.
- **Local Persistence** – Plans, sessions, and logs are saved to the browser automatically.

## Getting Started

1. Open `index.html` in any modern browser.
2. Customize the default plan or add your own days and exercises.
3. Use the Active Session tools to guide your workout and quick-log sets.
4. Record full session summaries in the journal to keep stats and charts up to date.

No build step or dependencies are required.
